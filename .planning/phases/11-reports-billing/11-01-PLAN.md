**Query functions — queries.ts:** Add two new exported async functions.

**`fetchReportsData(orgId: string, period: string): Promise<ReportsData | null>`**

1. Compute date ranges using a `getPeriodRange` helper:
   - `this_month`: start = 1st of current month, end = now, prev = same range for previous month
   - `last_month`: start = 1st of prev month, end = last day of prev month, prev = same range for month before
   - `last_3_months`: start = 90 days ago, end = now, prev = 90 days before that (180-91 days ago)
   - Return `{ start, end, prevStart, prevEnd }` as ISO strings

2. Fetch org automations (same pattern as fetchDashboardData — query automations where org_id = orgId, join template for avg_minutes_per_task and activity_metric_label):
   ```
   .from("automations").select("id, name, template:automation_templates(avg_minutes_per_task, activity_metric_label)").eq("organization_id", orgId)
   ```
   If no automations, return null (triggers empty state).

3. Guard: if `orgAutomationIds.length === 0`, return null.

4. Fetch executions for the FULL window (prevStart to end) in one query:
   ```
   .from("automation_executions").select("automation_id, started_at, status")
   .in("automation_id", orgAutomationIds)
   .eq("status", "success")
   .gte("started_at", prevStart)
   .lte("started_at", end)
   ```

5. Split executions in-memory into `periodExecs` (started_at >= start && <= end) and `prevExecs` (started_at >= prevStart && < start).

6. Compute KPI — tasks completed:
   - `tasksCompleted = periodExecs.length`
   - `prevTasks = prevExecs.length`
   - `tasksChange = prevTasks > 0 ? Math.round(((tasksCompleted - prevTasks) / prevTasks) * 100) : null`

7. Compute KPI — hours saved:
   - Build a Map from automationId → avg_minutes_per_task (from the automations query)
   - `totalMinutes = periodExecs.reduce((sum, e) => sum + (minutesMap.get(e.automation_id) ?? 0), 0)`
   - `hoursSaved = Math.round((totalMinutes / 60) * 10) / 10`
   - Same for prevExecs → `prevHours`
   - `hoursChange = prevHours > 0 ? Math.round(((hoursSaved - prevHours) / prevHours) * 100) : null`

8. Fetch org hourly_cost:
   ```
   .from("organizations").select("settings").eq("id", orgId).single()
   ```
   Extract: `const hourlyCost: number | null = (org?.settings as any)?.hourly_cost ?? null`
   Note: hourly_cost is integer DOLLARS (not cents) per Phase 07-03 decision.

9. Compute estimated value:
   - `estimatedValue = hourlyCost != null ? Math.round(hoursSaved * hourlyCost) : null`
   - For change: `prevValue = hourlyCost != null ? Math.round(prevHours * hourlyCost) : null`
   - `valueChange`: same percentage formula, null if prevValue is 0 or null

10. Compute 8-week chart — extend the existing `groupByWeek` pattern:
    Create a `groupBy8Weeks` helper (or modify groupByWeek with a `buckets` parameter):
    ```typescript
    function groupBy8Weeks(executions: { started_at: string }[], now: Date): WeeklyChartData[] {
      const buckets: WeeklyChartData[] = Array.from({ length: 8 }, (_, i) => ({
        week: `S${i + 1}`,
        count: 0,
      }));
      const nowMs = now.getTime();
      executions.forEach((exec) => {
        const daysAgo = (nowMs - new Date(exec.started_at).getTime()) / (24 * 60 * 60 * 1000);
        const bucket = Math.floor(daysAgo / 7);
        if (bucket >= 0 && bucket < 8) {
          buckets[7 - bucket].count += 1;
        }
      });
      return buckets;
    }
    ```
    Query last 56 days of SUCCESS executions for the chart (separate from period query — always 8 weeks regardless of selected period):
    ```
    .from("automation_executions").select("started_at")
    .in("automation_id", orgAutomationIds)
    .eq("status", "success")
    .gte("started_at", 56_days_ago_ISO)
    ```

11. Compute per-automation breakdown:
    Use `periodExecs` grouped by `automation_id`:
    ```typescript
    const breakdownMap = new Map<string, { name: string; metricLabel: string; count: number; minutes: number }>();
    for (const exec of periodExecs) {
      const auto = automationsMap.get(exec.automation_id);
      if (!auto) continue;
      if (!breakdownMap.has(exec.automation_id)) {
        breakdownMap.set(exec.automation_id, {
          name: auto.name,
          metricLabel: auto.template?.activity_metric_label ?? "",
          count: 0, minutes: 0,
        });
      }
      const entry = breakdownMap.get(exec.automation_id)!;
      entry.count += 1;
      entry.minutes += auto.template?.avg_minutes_per_task ?? 0;
    }
    const breakdown: AutomationBreakdownRow[] = [...breakdownMap.values()]
      .map((r) => ({
        automationId: r.name, // use as identifier in display
        name: r.name,
        metricLabel: r.metricLabel,
        count: r.count,
        hoursSaved: Math.round((r.minutes / 60) * 10) / 10,
      }))
      .sort((a, b) => b.hoursSaved - a.hoursSaved);
    ```
