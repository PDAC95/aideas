**resend-email-timer.tsx** — "use client" component:
- Props: `{ email: string }`
- State: `cooldown` (number, starts at 0), `sending` (boolean)
- On "Resend email" click:
  1. Set `sending = true`
  2. Call `resendVerificationEmail(email)` Server Action
  3. Set `sending = false`, start 60-second countdown
  4. setInterval decrements `cooldown` every second until 0
- Button text:
  - While sending: "Sending..." with spinner
  - While cooldown > 0: "Resend in {cooldown}s" (disabled)
  - Otherwise: "Resend verification email"
- Use `useTranslations('verifyEmail')` for button text
- The button starts enabled (user can resend immediately on first visit), then 60s cooldown after each send
