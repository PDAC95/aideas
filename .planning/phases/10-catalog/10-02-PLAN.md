    **Props received from RSC (all strings pre-resolved):**
    ```typescript
    interface CatalogClientProps {
      templates: Array<CatalogTemplate & { displayName: string }>;
      initialCategory: string;
      initialIndustry: string;
      initialSearch: string;
      translations: {
    ```

    **State management:**
    - `useState` for `category`, `industry`, `search` â€” initialized from props (NOT useSearchParams)
    - `useMemo` for filtered templates:
      - `category === "all"` â†’ all; `category === "mas_populares"` â†’ `t.is_featured === true`; else â†’ `t.category === category`
      - `industry === "all"` â†’ all; else â†’ `(t.industry_tags ?? []).includes(industry)`
      - `search === ""` â†’ all; else â†’ `t.displayName.toLowerCase().includes(search.toLowerCase())`
      - AND combination of all three filters

    **URL sync:**
    - `useRouter` from `next/navigation`
    - `updateParams` function: builds URLSearchParams, removes keys with value "all" or "", calls `router.replace('/dashboard/catalog?' + params.toString(), { scroll: false })`
    - Call updateParams on category/industry change; debounce search input (300ms) before updating URL

    **Render layout:**
    1. Title (`<h1>` with translations.title)
    2. Search input bar (text input with magnifying glass icon from lucide-react, translations.searchPlaceholder)
    3. Category tabs: horizontal scrollable row of buttons. Each button shows category label + count of templates matching that category (computed from full templates array, not filtered). Active tab has purple bg/text styling. Order: all, mas_populares, sales, marketing, customer_service, documents, productivity, reports, ai_agents.
    4. Industry chips: row of pill buttons below tabs. Active chip has purple outline. Include "Todas" as first chip. Order matches CONTEXT.md.
    5. Result count: "Showing X of Y" (substitute {filtered} and {total} in translations.subtitle)
    6. Grid: 3-column (lg:grid-cols-3 md:grid-cols-2 grid-cols-1) gap-4
       - Render CatalogCard for each filtered template
       - Pass pre-formatted price string: `translations.monthlyPrice.replace("{price}", formatPrice(t.monthly_price))`
    7. Empty state: when filtered.length === 0, show centered block with emoji or icon, translations.emptyTitle, and a button calling resetFilters (sets all to defaults) with translations.emptyCta

    **Price helper inside this file:**
    ```typescript

      new Intl.NumberFormat(locale, { style: "currency", currency: "USD", maximumFractionDigits: 0 }).format(cents / 100);
    ```
