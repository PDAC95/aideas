Create `/dashboard/automations/page.tsx` as an async Server Component:

1. **Auth guard**: `createClient() -> getUser()`, redirect to /login if no user
2. **Get org**: `getOrgId(user.id)`, handle no-org edge case
3. **Read filter**: `const { status } = await searchParams;` (Next.js 16 — searchParams is a Promise)
4. **Fetch data**: `fetchAutomationsPage(orgId)` — returns sorted automations
5. **Compute counts**: Count automations by status for tab badges: `{ all: total, active: N, in_setup: N, paused: N }`
6. **Filter**: If status param provided and valid, filter the sorted list. Keep full list for "all".
7. **Get translations**: `getTranslations("dashboard.automations")`
8. **Build status labels**: Map of status -> translated label (for StatusBadge)

**Render structure:**
```
<div>
  {/* Page header: title + count */}
  <div className="mt-4 mb-6">
    <h1 className="text-2xl font-bold">{t("title")}</h1>
    <p className="text-muted-foreground text-sm mt-1">
      {automations.length === 1 ? t("count_one", { count: total }) : t("count", { count: total })}
    </p>
  </div>

  {/* Filter tabs — wrap in Suspense because useSearchParams requires it */}
  <Suspense fallback={<div className="h-10 mb-4" />}>
    <AutomationsFilterTabs counts={counts} translations={filterTranslations} />
  </Suspense>

  {/* Cards grid or empty state */}
  {filteredAutomations.length === 0 ? (
    /* Empty state — different message for "no automations at all" vs "no matches for filter" */
    status && status !== "all" ? (
      <FilterEmptyState message={t(`emptyFilter.${status}`)} />
    ) : (
      <NoAutomationsEmptyState title={t("empty.title")} cta={t("empty.cta")} />
    )
  ) : (
    <div className="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">
      {filteredAutomations.map((automation) => (
        <AutomationCard
          key={automation.id}
          automation={automation}
          statusLabel={statusLabels[automation.status]}
          translations={cardTranslations}
          locale={locale}
        />
      ))}
    </div>
  )}
</div>
```

- Filter with no results: lighter empty state with contextual message and no CTA button

**Get locale**: Use `getLocale()` from next-intl/server to pass to AutomationCard for number formatting.

**Important:** The `searchParams` prop in Next.js 16 is `Promise<{ status?: string }>` — must await it.
